  • innerHTML , querySelector , addEventListener 의 정의와 사용법에 대해서 배웠다.

  • innerHTML 속성은 HTML content (innerHTML) of an element 의 값으로 리턴해준다.
    아래의 경우, <div>Hello!</div> 의 값을 버튼을 클릭함과 동시에 <h1>제목입니다.</h1> 로 업데이트 해준다.

  • JS 를 사용하다 보면 html의 클래스 이름 및 태그를 갖고 오기 위해서 querySelector 을 많이 사용한다고 한다.

기본형태:
element = baseElement.querySelector(selectors);

리턴되는 값은 인자를 받아 일치하는 baseElement의 자손의 엘리먼트를 갖고 온다.

querySelector 사용할때는 css-selector 로 2가지 태그 선택이 가능!
- . 클래스를 선택
- # 아이디를 선택

아래의 경우, first element 인 <div>Hello!</div> 를 선택

  • EventTarget.addEventListener() 는 이벤트를 등록하는 가장 권장되는 방식이다. 이 방식을 이용하면 여러개의 이벤트 핸들러를 등록할 수 있다고 한다.

220418_TIL - HTML Outline 속성

Posted on 2022-04-18 Edited on 2022-04-20 In TIL (Today I Learned)

  • 블로그 내에 드디어 카테고리와 태그를 만들었다! 여러 에러 끝에 결국 테마를 바꿔야 했지만…

  • HTML 요소의 가장 바깥 부분을 둘러싸고 있는 Outline 부분의 스타일 속성에 대해서 배웠다.
    이 속성은 border 속성과는 달리 html 요소의 전체 크기에는 포함되지 않아 width, height 들은 아웃라인의 두께에 전혀 영향을 받지 않는다.

  • <h1>, <h6> 태그의 실제 사용 용도와 우선순위에 대해서 배웠다.

웹사이트를 처음 만들때 웹페이지의 어떤 특정 부분이 검색엔진에 잘 노출될지 그 중요도를 고려해 가장 중요한 <h1> 태그 순으로 <h4> 까지 보통 사용한다고 한다.

이러한 태그 요소들이 html 아웃라인과 연관이 있고 이러한 관점에서 보기 위해 다른 웹사이트들의 구조를 잘 참고해서 안목을 길러봐야 겠다.
